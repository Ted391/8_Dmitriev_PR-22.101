О сайте

Добро пожаловать на мой сайт D2K. Название моего сайта происходит от сокращения моего псевдонима DER|-2SH-|KA, который я использую для обозначения своих продуктов. Этот сайт представляет собой первоисточник для скачивания и использования продуктов, созданных DER|-2SH-|KA. Все продукты бесплатны и не требуют активации. Однако автор был бы очень рад вашей материальной поддержке. Больше информации о поддержке в разделе "Обо мне".

Первая программа - WePYP

WePYP - это программа, созданная для удобного хранения, изменения, копирования и удаления паролей. Программа написана на динамическом объектно-ориентировочном языке программирования Python. Для создания UI интерфейса была использована библиотека PyQt6, для создания базы хранения паролей SQLite3 и для внутренних настроек библиотека JSON.
