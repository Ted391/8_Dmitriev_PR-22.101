WePYP

Версия: 1.3.2

Размер: 54.3 Мбайт

Тип файла: .exe

Поддержка: Windows 10 / 11

Доступ: Бесплатно

WePYP - это программа, созданная для удобного хранения, изменения, копирования и удаления паролей. Программа написана на динамическом объектно-ориентировочном языке программирования Python. Для создания UI интерфейса была использована библиотека PyQt6, для создания базы хранения паролей SQLite3 и для внутренних настроек библиотека JSON.
